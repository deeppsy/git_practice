Let's get down to it

Here is a table describing all the table tags in HTML tables

important table tags

No Tag Tag Name Description
1 <table> table This creates the table
2 <thead> Table head The headings of the table usually go here
3 <tbody> Table body The body of the table usually go here
4 <tfoot> Table footer The footer of the table usually go here. For example, doing a spreadsheet that has a row for "Total"
5 <td> Table data This allow us input data into the cells
6 <tr> Table row This creates a row for you.
7 <th> Table head This is usally used to input the headings not the td.

important table attributes

Here is a table describing the important table attributes I don't want to forget

No Attribute Name Description
1 <colspan> Column Span This helps extend the no of column the th or td tag extends to.
2 <rowspan> Row Span This helps extend the no of row the th or td tag extends to.
3 scope scope This enumerated attribute defines the cells that the "th" element relates to. It may have the following values:
row: The header relates to all cells of the row it belongs to.
col: The header relates to all cells of the column it belongs to.
